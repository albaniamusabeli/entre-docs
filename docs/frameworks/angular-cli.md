# Comandos para la instalación de Angular CLI

## Iniciar un proyecto

```bash
ng new <nombre-proyecto>
```

## Generar Módulos

::: tip Opciones
* --routing: agrega archivo de routing
:::

```bash
ng g m <nombre-modulo> --routing
```

## Generar Componentes

```bash
ng g c <nombre-componente>
```

## Generar Servicios

```bash
ng g s <nombre-servicio>
```
