# Comandos de Angular CLI

## Iniciar un proyecto

```bash
ng new <nombre-proyecto>
```

## Generar MÃ³dulos

::: tip Opciones
* --routing: agrega archivo de routing
:::

* Global
```bash
ng g m <nombre-modulo> --routing
```

* Local
```bash
npm run ng -- g m <nombre-modulo> --routing 
```

## Generar Componentes

* Global
```bash
ng g c <nombre-componente>
```

* Local
```bash
npm run ng -- g c <nombre-componente>
```

## Generar Servicios

* Global
```bash
ng g s <nombre-servicio>
```

* Local
```bash
npm run ng -- g s <nombre-servicio>
```
